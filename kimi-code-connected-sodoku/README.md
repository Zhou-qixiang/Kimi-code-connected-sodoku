# 连通数独

一个纯前端实现的「连通数独」益智小游戏，玩法参考《最强大脑》第十三季连通数独项目：
**四个 6×6 数独区域十字交叉组成一个统一大棋盘，区域之间真实重叠，重叠格必须同时满足两个区域的全部规则。**

每次开局自动生成新题，保证**唯一解**。

---

## 如何运行

无需安装任何东西，也无需服务器：

> **双击 `index.html`，用浏览器打开即可开始游戏。**

文件结构：

```
/
├── index.html   ← 界面结构
├── style.css    ← 样式（暗色主题）
├── script.js    ← 游戏逻辑 + 题目生成器 + 唯一解求解器
└── README.md
```

也可以通过网址参数直达某个难度：`index.html?diff=easy`（`easy` / `medium` / `hard`）。

## 棋盘长什么样

整个棋盘是一个 14×14 的统一坐标系，四个 6×6 区域按十字交叉排布：

```
            ┌─────────┐
            │  奇偶数独 │（蓝框）
            └────╮    ╭────┐
   ┌─────────┐  │ 差值数独│（绿框）
   │  彩色数独 │（紫框）│    │
   └────╮    ╭──┴────┘
        │  箭头数独 │（橙框）
        └─────────┘
```

- 相邻区域在 **2×2 角部真实重叠**（共 16 个重叠格）
- 重叠格带有**斜纹标识**：它同时属于两个区域，填的数字必须同时满足两个区域的行、列、宫约束和特殊规则
- 这正是"连通"的含义：一个区域的推理会顺着重叠格传导到另一个区域

## 游戏规则

### 基本规则（对所有区域生效）

每个 6×6 区域中填入数字 **1～6**：

1. 每行数字不重复
2. 每列数字不重复
3. 每个 **2×2 宫**数字不重复（每个 6×6 区域划分为 9 个 2×2 小宫）

### 特殊规则（每个区域各有一种）

#### ① 奇偶数独（蓝色区域）
两个格子之间有**白点 ●** 时，这两格必须**一奇一偶**。

```
3 ● 4   ✔ 正确（奇 + 偶）
3 ● 5   ✘ 错误（奇 + 奇）
```

#### ② 差值数独（绿色区域）
两个格子之间有**绿圈数字** 时，该数字 = 两格数字之差的绝对值。

```
2 ③ 5   ✔ 正确（|5 − 2| = 3）
2 ④ 5   ✘ 错误（|5 − 2| = 3 ≠ 4）
```

#### ③ 彩色数独（紫色区域）
整个区域被划分为若干**连通的彩色块**：同一颜色块内数字不重复。
（颜色块都是真正相连的格子，不会隔空出现）

#### ④ 箭头数独（橙色区域）
每个 2×2 宫内有一个**橙色斜箭头**（↘ ↙ ↗ ↖ 之一），
**箭头指向的格子 = 该 2×2 宫内的最大数字**。

```
1 3
2 6     箭头 ↘ → 指向 6（最大）
```

### 重叠格（斜纹格）

斜纹格同时属于两个区域，例如奇偶区与彩色区的重叠格，既要满足白点约束，又要满足所在颜色块不重复的约束。**它是破解整道题的关键突破口。**

## 操作方法

| 操作 | 方式 |
|------|------|
| 选中格子 | 鼠标点击 |
| 填数字 | 点击下方数字按钮 1～6，或按键盘 `1`～`6` |
| 擦除 | 点「橡皮擦」，或按 `Delete` / `Backspace` |
| 候选数字 | 点「候选数字」进入笔记模式，再填的数字会变成小字候选标记（再点一次退出） |
| 移动选格 | 键盘方向键（自动跳过棋盘空缺处） |
| 检查答案 | 点「检查答案」：行/列/宫重复和特殊规则错误会**标红**提示 |
| 重新生成 | 点「换一题」按当前难度生成新题 |
| 重来本题 | 点「重置」清空自己填的所有数字 |

- **计时器**在顶栏自动计时。
- 全部格子填满且无任何错误时，自动弹出胜利动画并显示用时。

## 难度说明

三个档位点击后都会**自动生成一道新题**（约 0.2 秒），区别主要在已知数字的数量：

| 难度 | 已知数（共 128 格） | 适合 |
|------|--------------------|------|
| 简单 | 56 | 新手入门，熟悉四种规则 |
| 中等 | 44 | 需要结合重叠格做跨区域推理 |
| 困难 | 34 | 大量留白，需要候选数 + 多规则联动 |

> 出题流程：先生成满足全部规则的完整答案 → 按答案布置白点/差值/颜色块/箭头 → 逐个挖掉数字，**每挖一格都由内置求解器验证全题仍有唯一解**，否则放回。因此每道题都一定有且只有一个答案。

---

## 新手小提示

1. **从重叠格入手**：重叠格受两个区域约束，候选范围最小。
2. **箭头是最强线索**：箭头直接告诉你 2×2 宫里最大值的位置，先填 5、6 这类大数。
3. **彩色块结合行列**：颜色块常与行/列重叠，可以快速排除重复。
4. 卡住时打开「候选数字」做笔记，或点「检查答案」确认目前没走错。
